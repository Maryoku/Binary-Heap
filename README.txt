Кучи, в частности их двоичная разновидность - весьма популярные структуры данных. 
Пользуясь своими познаниями в generic’ах, реализуйте бинарную кучу на базе массива, которая бы хранила значения подходящих куче типов.

Сделайте класс MaxBinaryHeap со следующей функциональностью:
1. Конструктор должен создавать объект кучи с максимальным корнем с элементами заданного generic типа, без вершин. 
2. Метод Insert позволяет вставить новое значение корректного типа в кучу (приводит к перестроению).
3. Метод Extract позволяет извлечь корневое значение корректного типа из кучи (приводит к перестроению).
